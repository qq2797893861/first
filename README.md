# 校园闲置好物交换站

一个基于 React + TypeScript + Vite 的校园闲置物品交换平台。

## 功能特性

- 🏠 **首页** - 分类导航和最新发布
- 🔍 **发现页** - 搜索和物品筛选
- 📝 **发布页** - 发布闲置物品
- 👤 **个人中心** - 登录、我的物品、收藏
- 💬 **消息** - 联系卖家
- 📦 **物品详情** - 查看物品信息、收藏

## 技术栈

- React 18 + TypeScript
- Vite 5
- Tailwind CSS
- Zustand（状态管理）
- React Router v6（路由）
- Lucide React（图标）

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
